# Módulo 17 — Segurança e Comportamento

## Sobre este módulo

JavaScript tem comportamentos que podem surpreender (e causar bugs). Entender **strict mode**, **coerção de tipos** e a diferença entre `==` e `===` é fundamental para escrever código previsível e seguro.

## O que você vai aprender

- O que o strict mode muda e por que usar
- Como a coerção de tipos funciona (implícita e explícita)
- A diferença real entre `==` e `===`
- Como evitar armadilhas comuns
