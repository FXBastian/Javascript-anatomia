# Módulo 04 — Estruturas de Dados

## Sobre este módulo

Objetos e Arrays são as duas estruturas de dados fundamentais do JavaScript. Praticamente **tudo** na linguagem é construído sobre eles. Dominar a criação, acesso e manipulação dessas estruturas é essencial para qualquer aplicação real.

## O que você vai aprender

- Como criar e manipular objetos (propriedades, métodos, iteração)
- Como criar e manipular arrays (métodos mutáveis e imutáveis)
- Métodos mais importantes e padrões de uso profissional
