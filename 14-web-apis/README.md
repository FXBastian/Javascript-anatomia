# Módulo 14 — Web APIs

## Sobre este módulo

As **Web APIs** são fornecidas pelo navegador (não pelo JavaScript em si). São ferramentas essenciais para comunicação com servidores, armazenamento local e controle de tempo. Todo projeto real usa pelo menos uma delas.

## O que você vai aprender

- Fetch API para requisições HTTP
- LocalStorage e SessionStorage para persistência local
- setTimeout, setInterval e requestAnimationFrame
