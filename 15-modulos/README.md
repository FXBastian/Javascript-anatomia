# Módulo 15 — Módulos

## Sobre este módulo

Módulos permitem **organizar código** em arquivos separados, cada um com suas responsabilidades. O sistema de módulos ES6 (`import`/`export`) é o padrão moderno e usado em todos os frameworks atuais.

## O que você vai aprender

- Como exportar e importar valores entre arquivos
- Named exports vs default exports
- Organização de projetos com módulos
- Dynamic imports para carregamento sob demanda
