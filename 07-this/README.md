# Módulo 07 — This (Contexto)

## Sobre este módulo

O `this` é uma das partes mais confusas do JavaScript para iniciantes. Ele **não é fixo** — seu valor depende de **como** a função é chamada, não de onde ela é definida (exceto em arrow functions). Este módulo elimina essa confusão de vez.

## O que você vai aprender

- As 4 regras que determinam o valor de `this`
- A diferença fundamental entre `this` em functions e arrow functions
- Como `bind`, `call` e `apply` alteram o `this`
- Armadilhas comuns e como evitá-las
