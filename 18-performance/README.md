# Módulo 18 — Performance e Memória

## Sobre este módulo

Entender como JavaScript gerencia memória é o que separa desenvolvedores juniores de seniores. Saber a diferença entre **valor e referência**, como o **Garbage Collector** funciona e como evitar **memory leaks** é essencial para aplicações de qualidade.

## O que você vai aprender

- Garbage Collection: como a engine libera memória automaticamente
- Tipos por valor vs por referência
- Como memory leaks acontecem e como evitá-los
- Técnicas de otimização de performance
