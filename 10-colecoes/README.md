# Módulo 10 — Coleções Avançadas

## Sobre este módulo

Além de objetos e arrays, JavaScript oferece estruturas de dados especializadas: **Map**, **Set**, **WeakMap** e **WeakSet**. Cada uma resolve problemas específicos que objetos e arrays não resolvem bem.

## O que você vai aprender

- Map: chave-valor com qualquer tipo de chave
- Set: coleção de valores únicos
- WeakMap e WeakSet: versões com referências fracas (garbage collection friendly)
- Quando usar cada estrutura
