# Módulo 09 — Estruturas Modernas (ES6+)

## Sobre este módulo

O ECMAScript 2015 (ES6) foi a maior atualização da história do JavaScript. As features introduzidas aqui são **padrão na indústria** e usadas em todo projeto moderno. Saber usá-las com fluência é requisito básico para qualquer vaga.

## O que você vai aprender

- Destructuring de objetos e arrays
- Operadores Spread e Rest
- Template Literals para strings dinâmicas
- Optional Chaining e Nullish Coalescing
