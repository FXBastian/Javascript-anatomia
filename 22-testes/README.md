# Modulo 22 — Testes

## Sobre este modulo

Codigo sem testes e codigo que voce nao confia. Este modulo ensina a escrever testes que dao confianca para refatorar, adicionar features e deployar sem medo.

## O que voce vai aprender

- Tipos de testes: unitarios, integracao, e2e
- Anatomia de um teste: Arrange, Act, Assert
- Mocks, stubs e spies
- Code coverage e o que ele realmente significa

## Arquivos deste modulo

| Arquivo | Conteudo |
|---------|----------|
| [unitarios.md](unitarios.md) | Testes de funcoes isoladas, AAA pattern |
| [mocks.md](mocks.md) | Simular dependencias: mocks, stubs, spies |
