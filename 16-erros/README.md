# Módulo 16 — Tratamento de Erros

## Sobre este módulo

Erros acontecem. A diferença entre um programa amador e profissional é **como os erros são tratados**. JavaScript oferece mecanismos robustos para capturar, tratar e lançar erros de forma controlada.

## O que você vai aprender

- Como usar `try`, `catch` e `finally`
- Como lançar erros personalizados com `throw`
- Tipos de erro nativos (TypeError, ReferenceError, etc.)
- Tratamento de erros em código assíncrono
