# Modulo 20 — Arquitetura e Design Patterns

## Sobre este modulo

Design Patterns sao solucoes reutilizaveis para problemas recorrentes no desenvolvimento de software. Em JavaScript, muitos patterns ganham formas unicas gracas a funcoes de primeira classe, closures e prototipos.

Este modulo transforma voce de alguem que "escreve codigo que funciona" para alguem que "projeta codigo que escala".

## O que voce vai aprender

- O que sao Design Patterns e por que importam
- Module Pattern e suas variacoes
- Observer Pattern (pub/sub)
- Factory, Singleton, Strategy, Decorator
- Patterns modernos com classes e modulos ES6+
