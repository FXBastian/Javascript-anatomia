# Módulo 02 — Controle de Execução

## Sobre este módulo

Controle de execução é o que dá **lógica** ao seu programa. Sem ele, o código seria apenas uma sequência linear de instruções. Aqui você aprende a tomar decisões, repetir ações e controlar o fluxo.

## O que você vai aprender

- Como usar condicionais para tomar decisões (`if`, `else`, `switch`)
- Como repetir blocos de código com loops (`for`, `while`, `do...while`, `for...in`, `for...of`)
- Como interromper ou pular iterações com `break`, `continue` e `return`
