# Modulo 21 — Node.js e Ambiente de Execucao

## Sobre este modulo

JavaScript nao vive apenas no navegador. Com Node.js, a linguagem ganhou acesso ao sistema operacional: arquivos, rede, processos e muito mais. Este modulo explora JavaScript fora do browser.

## O que voce vai aprender

- Como Node.js funciona internamente (V8 + libuv)
- Criar um servidor HTTP do zero
- Ler e escrever arquivos de forma sincrona e assincrona
- Diferenca entre browser e Node.js
