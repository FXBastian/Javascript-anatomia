# Módulo 13 — Manipulação de DOM

## Sobre este módulo

O **DOM (Document Object Model)** é a interface que conecta JavaScript ao HTML. Toda interatividade em páginas web passa por aqui. Dominar o DOM é essencial para qualquer desenvolvedor frontend.

## O que você vai aprender

- Como selecionar elementos no HTML
- Como escutar e responder a eventos do usuário
- Como alterar conteúdo, estilos e atributos dinamicamente
