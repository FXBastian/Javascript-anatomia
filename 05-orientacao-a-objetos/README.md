# Módulo 05 — Orientação a Objetos

## Sobre este módulo

JavaScript é uma linguagem **baseada em protótipos**, não em classes tradicionais. A sintaxe `class` do ES6 é açúcar sintático sobre o sistema de protótipos. Aqui você entende ambos os modelos e como eles se relacionam.

## O que você vai aprender

- Como criar classes com construtores, métodos e propriedades
- Como funciona herança com `extends` e `super`
- O sistema de protótipos (prototype chain)
- Encapsulamento com campos privados
