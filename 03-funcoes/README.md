# Módulo 03 — Funções

## Sobre este módulo

Funções são o **coração do JavaScript**. São cidadãs de primeira classe — podem ser atribuídas a variáveis, passadas como argumento e retornadas de outras funções. Entender funções profundamente é o que separa um iniciante de um programador competente.

## O que você vai aprender

- Como declarar funções de diferentes formas
- Como parâmetros e retorno funcionam
- Arrow functions e suas diferenças fundamentais
- Funções de ordem superior, closures e padrões avançados
