# Módulo 12 — Built-in Objects

## Sobre este módulo

JavaScript vem com objetos nativos poderosos para manipulação de strings, números, datas, matemática e JSON. Conhecer esses objetos evita reinventar a roda e permite escrever código mais limpo e eficiente.

## O que você vai aprender

- Métodos de String para busca, transformação e extração
- Métodos de Number para formatação e validação
- Math para cálculos matemáticos
- Date para manipulação de datas e horários
- JSON para serialização e desserialização de dados

## Arquivos deste módulo

| Arquivo | Conteúdo |
|---------|----------|
| [string.md](string.md) | Métodos de string, regex básico, template literals |
| [number.md](number.md) | parseInt, parseFloat, toFixed, isNaN, Number.isFinite |
| [math.md](math.md) | Math.random, floor, ceil, round, pow, min, max |
| [date.md](date.md) | new Date, métodos get/set, formatação, Intl.DateTimeFormat |
| [json.md](json.md) | JSON.parse, JSON.stringify, reviver, replacer |
