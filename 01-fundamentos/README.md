# Módulo 01 — Fundamentos da Linguagem

## Sobre este módulo

Este é o ponto de partida. Aqui você aprende **como o JavaScript se escreve**, quais são seus blocos básicos de construção e como a linguagem organiza dados e operações.

Sem dominar os fundamentos, todo o resto será construído sobre uma base frágil.

## O que você vai aprender

- Como a sintaxe do JavaScript é estruturada (palavras reservadas, ponto e vírgula, comentários)
- Quais são os tipos de dados primitivos e estruturados
- Como declarar e usar variáveis com `var`, `let` e `const`
- Todos os operadores da linguagem e como eles funcionam
